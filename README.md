Good for debugging webhooks, locally.

To install you will need [node](http://nodejs.org/) 0.4.x and [npm](http://npmjs.org/) 1.x.

    git clone https://github.com/hecticjeff/growler
    cd growler
    npm install
    ./node_modules/.bin/coffee index.coffee
